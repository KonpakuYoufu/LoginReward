# LoginReward
This is a plugin for Grasscutter .

It allows you to reward players when they login every day .

### Features
You can configure the mail with:
<ul>
<li>Title</li>
<li>Content</li>
<li>Sender</li>
<li>Expiry time (in seconds)</li>
<li>Important (marked with a star)</li>
<li>Attachment</li>
</ul>
